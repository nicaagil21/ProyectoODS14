# 🌊 CRUD – Juego ODS 14 (Limpieza del Océano)

## 📖 Descripción
Este proyecto consiste en la implementación de un **CRUD (Crear, Leer, Actualizar y Borrar)** relacionado con un juego basado en el **ODS 14: Vida submarina**.

En esta primera fase no se desarrolla el juego, sino la gestión de datos que se utilizarán en él (por ejemplo, basura, puntuaciones o elementos del juego).

---

## 🎯 Objetivo
- Practicar operaciones CRUD con JavaScript  
- Organizar datos que se usarán en el juego  
- Preparar la base del proyecto final  

---

## 🛠️ Tecnologías
- HTML  
- CSS  
- JavaScript (sin frameworks)  

---

## ⚙️ Funcionalidades CRUD
El sistema permite:

- ➕ Crear nuevos elementos (ej: tipo de basura)  
- 📄 Leer los elementos existentes  
- ✏️ Actualizar datos  
- ❌ Eliminar elementos
